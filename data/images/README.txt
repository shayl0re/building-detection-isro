Place your satellite images (.JPG) here.

Each image should have a corresponding mask with the same name in the data/masks/ folder.

Example structure:
  data/images/triple_0.JPG   ←→   data/masks/triple_0.PNG
  data/images/triple_1.JPG   ←→   data/masks/triple_1.PNG
  ...

Dataset used in this project:
  - 200 samples from ISRO RRSC-South, Bengaluru
  - Image size: 250 × 250 pixels (RGB)
  - Mask size:  250 × 250 pixels (Grayscale)

Note: The actual dataset files are not included in this repository
due to size constraints. Please contact the author or use a publicly
available dataset such as:
  - SpaceNet: https://spacenet.ai/
  - Inria Aerial Image Labeling Dataset: https://project.inria.fr/aerialimagelabeling/
